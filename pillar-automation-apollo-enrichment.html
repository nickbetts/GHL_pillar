7. Apollo Enrichment

Enriches contacts with company/person intelligence and optional ICP bonus.

Related Webhooks

Optional from ContactCreate flow

Trigger Mode

  • POST /api/apollo with contactId.
  • Skips when APOLLO_API_KEY is missing or contact lacks email/company.

Actual Field Mapping Rules

  • linkedin_url -> LinkedIn Profile URL
  • title -> title
  • annual_revenue bands -> Annual Income Band: >1,000,000 => £1m+, >250,000 => £250k–£1m, >50,000 => £50k–£250k, else <£50k
  • employee_count bands -> Org Size Band: >500 => £1m+, >200 => £250k–£1m, >50 => £50k–£250k, else <£50k

Actual ICP Bonus Rules

  • +20 if title matches VP/C-level/Chief/President/Director AND annual_revenue > 1,000,000.
  • +15 if employee_count is between 50 and 200 (inclusive) AND industry contains non-profit.
  • Final ICP Score update is clamped with Math.min(current + bonus, 100).
Implementation source: api/apollo.js. Response includes enriched fields and icpBonus.
