4. Assignment Router

Routes opportunities to the right owner based on persona and stage.

Related Webhooks

OpportunityCreate OpportunityUpdate

Exact Steps

  • Receive opportunity event and read stage + assignedTo.
  • Run only when stage is Lead and assignedTo is empty.
  • Fetch related contact and read Governance Role persona.
  • Resolve owner from persona mapping; fallback to default owner.
  • Update opportunity assignedTo with resolved owner ID.

Actual Routing Rules

  • If stage != "Lead" -> no assignment action.
  • If assignedTo already exists -> no assignment action.
  • Persona = "Fundraising Director" -> GHL_FUNDRAISING_OWNER (or default).
  • Persona = "Operations Manager" -> GHL_OPS_OWNER (or default).
  • Any other persona -> GHL_DEFAULT_OWNER.
