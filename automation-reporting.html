6. Reporting

Builds daily and weekly operating metrics from live contact and opportunity data.

Related Webhooks

No direct webhook trigger

Trigger Mode

  • Manual API call to /api/reports with type query.
  • Recommended scheduled trigger from external scheduler or workflow runner.

Exact Steps

  • Read contacts and opportunities via GHL API-safe query limits.
  • Aggregate key metrics for daily or weekly output.
  • Build response payload with timestamp and metric map.
  • Optional email dispatch path can be enabled via service integration.
  • Return report JSON to caller for dashboarding or archive.
This automation is operational reporting, not event-webhook orchestration.
