7. Apollo Enrichment

Enriches contacts with company/person intelligence and optional ICP bonus.

Related Webhooks

Optional from ContactCreate flow

Trigger Mode

  • Manual POST to /api/apollo with contactId.
  • Can be called from ContactCreate workflow if desired.

Exact Steps

  • Validate APOLLO_API_KEY and incoming contactId.
  • Fetch contact from GHL and verify email/company fields exist.
  • Send lookup request to Apollo contacts search API.
  • Map Apollo fields to GHL custom field names.
  • Calculate ICP bonus where enrichment thresholds are met.
  • Write enriched fields and score update back to GHL.
If APOLLO_API_KEY is absent, endpoint safely returns skipped/not_found behavior.
