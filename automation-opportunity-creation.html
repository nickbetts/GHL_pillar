3. Opportunity Creation

Creates opportunities when qualification thresholds are reached.

Related Webhooks

ContactCreateContactUpdate

Exact Steps

  • Smart tagging adds high-intent tag to contact when rules match.
  • System checks existing opportunities for that contact.
  • If no open opportunities exist, new opportunity payload is prepared.
  • Payload includes name, pipeline, stage, persona, org size, and source channel.
  • Opportunity is created in GHL with Lead-stage defaults.
  • Creation event then triggers assignment router automation.
Requires GHL_PIPELINE_ID and default owner settings in env.
