3. Opportunity Creation

Creates opportunities when qualification thresholds are reached.

Related Webhooks

ContactCreate ContactUpdate

Exact Steps

  • Trigger only when smart tagging adds high-intent in current run.
  • Query existing opportunities by contactId.
  • If no existing opportunities, build create payload and call createOpportunity.
  • Create with stage = Lead and assignedTo = GHL_DEFAULT_OWNER.
  • OpportunityCreate event then activates assignment router.

Actual Create Conditions and Payload

  • Condition: tagsToAdd.includes("high-intent") must be true.
  • Duplicate guard: existingOpps.data must be empty.
  • Name: "{companyName or New} - {firstName} {lastName}"
  • Pipeline: process.env.GHL_PIPELINE_ID
  • Stage: "Lead"
  • AssignedTo: process.env.GHL_DEFAULT_OWNER
  • CustomFields.Persona = contact["Persona"]
  • CustomFields.Org Size Band = contact["Annual Income Band"]
  • CustomFields.Lead Source Channel = contact["UTM Source"] or "Direct"
