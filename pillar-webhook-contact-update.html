Webhook: ContactUpdate

This event is triggered whenever an existing contact changes in GHL.

Function in CRM

ContactUpdate keeps lead intelligence accurate over time. It re-runs scoring and tagging every time meaningful data changes.

  • Refreshes ICP score as budget, timeline, role, or engagement changes.
  • Adds newly applicable tags from current contact state.
  • Can create an opportunity if updates now qualify lead as high intent.
  • Prevents stale segmentation and improves routing quality.
How it works with CRM: this turns your contact profile into a living model, not a one-time snapshot.
