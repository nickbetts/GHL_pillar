8. Deduplication

Detects and merges duplicate contacts using exact and fuzzy matching.

Related Webhooks

No direct webhook trigger

Trigger Mode

  • Manual POST to /api/dedup for duplicate scan.
  • Manual POST to /api/dedup?action=merge for merge operation.
  • Recommended scheduled run with operator review.

Exact Steps

  • Load contacts from GHL up to API-safe limits.
  • Find exact email duplicate groups.
  • Find exact phone duplicate groups.
  • Run fuzzy name+company similarity checks for potential duplicates.
  • Return structured duplicate candidates with confidence level.
  • On merge action, combine tags/notes and mark secondary as merged.
Merge is intentionally explicit to keep operator control over data integrity.
