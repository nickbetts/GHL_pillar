Webhook: OpportunityUpdate

This event is triggered whenever an opportunity changes stage, value, or ownership.

Function in CRM

OpportunityUpdate keeps pipeline operations coherent. It is currently lightweight but acts as the extension point for stage-based actions.

  • Receives stage transitions and updated opportunity payloads.
  • Supports owner reassignment checks when stage returns to Lead.
  • Can be expanded for tasks, notifications, and SLA timers.
  • Provides clean trigger point for future expansion and churn workflows.
How it works with CRM: this is the orchestration hook for stage automation as your sales process matures.
