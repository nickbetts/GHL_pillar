8. Deduplication

Detects and merges duplicate contacts using exact and fuzzy matching.

Related Webhooks

No direct webhook trigger

Trigger Mode

  • POST /api/dedup for scan mode.
  • POST /api/dedup?action=merge with primary and secondary IDs for merge mode.

Actual Duplicate Detection Rules

  • Exact email match groups with confidence = high.
  • Exact phone match groups with confidence = high.
  • Fuzzy name+company match when companyName exactly matches and Levenshtein similarity > 85% (confidence = medium).
  • Primary record chosen as most recently created in each duplicate group.

Actual Merge Behavior

  • Primary tags become union(primary.tags, secondary.tags).
  • Primary notes append secondary notes with merge marker.
  • Secondary record is marked with note MERGED INTO {primaryId} and tag merged.
Implementation source: api/dedup.js. Scan and merge are intentionally separated for operator control.
