9. Webhook Orchestration

Central router that dispatches incoming events to the correct automation handlers.

Related Webhooks

ContactCreateContactUpdateOpportunityCreateOpportunityUpdateInboundMessage

Exact Steps

  • Receive POST payload at /api/webhooks.
  • Validate method and parse event type from body.
  • Dispatch to contact, opportunity, or inbound-message handler branch.
  • Execute branch automations and catch per-handler errors.
  • Return structured status payload to webhook caller.
  • Log event type and result for runtime diagnostics.
This automation is the control layer; all event-driven behaviors depend on it.
