9. Webhook Orchestration

Central router that dispatches incoming events to the correct automation handlers.

Related Webhooks

ContactCreate ContactUpdate OpportunityCreate OpportunityUpdate InboundMessage

Exact Dispatch Rules

  • ContactCreate and ContactUpdate -> handleContactEvent(data).
  • OpportunityCreate and OpportunityUpdate -> handleOpportunityEvent(data).
  • InboundMessage -> handleInboundMessage(data).
  • Unknown type -> status skipped with reason Unhandled event type.

Exact Runtime Flow

  • Reject non-POST requests with HTTP 405.
  • Wrap event handling in try/catch and return HTTP 500 on failure.
  • Log every incoming type and handler result for diagnostics.
  • Return structured JSON status for each processed event.
Implementation source: api/webhooks.js. This is the control plane for all event-driven automations.
