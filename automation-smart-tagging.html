2. Smart Tagging

Applies dynamic tags from persona, sector, engagement, and pain-point signals.

Related Webhooks

ContactCreateContactUpdate

Exact Steps

  • After contact fetch and scoring, current tags are loaded.
  • 8 rule sets evaluate contact fields and activity state.
  • Missing tags are computed and merged with existing tags.
  • Tag list is deduplicated to prevent duplicate writes.
  • Contact record is updated only if net-new tags exist.
  • Downstream triggers (for example high-intent) are evaluated after tagging.
Output of this automation drives opportunity creation and nurture segmentation.
